Welcome!

This page will take you through the process of building a small script to pull, save, and visualize data from TD Ameritrade's API.

You'll be doing most of the work yourself! I've tried my best to exclude as much code as possible while still making the walkthrough easy to follow. However, the guide is probably not perfect, so if I've used any terms you aren't sure about, or if you need any additional explanation about a topic, please don't hesistate to ask me about it. This project touches on a lot of different areas, so I'd expect that there will be quite a few questions.

Our main goals are to:

  • Collect daily data of our portfolio's value and positions
  • Automate the process of data collection as much as possible
  • Generate a graph to visualize the data
